Unofficial Official Path of Exile API: Character Skill Tree

Last Updated: 2015/12/29

This page contains unofficial documentation on the pathofexile.com API to obtain the passive skill tree for a character. This is not strictly an official API service, however it is publicly available and used by many tools.

This API is available at the following base URL:

https://www.pathofexile.com/character-window/get-passive-skills

Request:

This API accepts the following parameters via GET:

Parameter Description
accountNameREQUIRED: The name of the account that owns the character whose skill tree you would like to see
characterREQUIRED: The name of the character whose skill tree you would like to see
characterREQUIRED: The name of the character whose skill tree you would like to see
reqData[ 0 | 1 ]

If set to 1 the entire skill tree data is appended to the output. This should rarely be necessary. It is recommended to keep this to the default of 0
It is important to note that this API will only return data for accounts with character privacy disabled, or your own account if you have a valid session id!

Response:

All characters are returned in a simple JSON array, with each character embedded as a JSON object:

JSON field Description
nameThe character's name
leagueThe league the character is in
classId

A numerical id indicating the class:

0
Scion
1
Marauder
2
Ranger
3
Witch
4
Duelist
5
Templar
6
Shadow
classText of the class type
levelThe current level of the character

Invalid account names or accounts with characters hidden via privacy settings will return the generic response:

false
