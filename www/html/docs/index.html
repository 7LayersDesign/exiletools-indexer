Pete's Path of Exile APIs

I've created and released these API's to help tool builders who want to add value to the community without needing to have servers set up to collect and store data.

All of these API's are provided free, with my servers, processing, scripts, and databases as the backend. None of these API's make direct calls to any systems other than my own unless specifically stated. No guarantees are provided about availability, uptime, etc. - please don't try to monetize a system based on my API's.

Stash Tab Indexer ElasticSearch API

    This API provides direct access to my Elastic Search Index for Public Stash Tabs, updated in near-realtime (usually within 10s) as changes are posted through the official API at pathofexile.com. It can be used for everything from creating a poe.trade style search to alerting on specific item availability to price reports to just finding out whether or not you're getting a fair price for an item to even creating advanced reports on items, prices, and availability.


    Production use of this API requires you to sign up for an API Key: There is a development API key you can use initially, but I'd like to keep track of who is using this and have a way to contact you, so please sign up for an API key. More information is available on the API page.

    Some tools that use my Shop Indexer ElasticSearch API include:

    • In-Game Price Comparison Macro
    • Real Time Unique Item Availability
    • Real Time Item Availability of All Items
    • Monitor an Account's Stash Tabs in Real Time
    • Unique Item Availability and Price Reports
    • Streaming socket access to the indexer Log File

Ladder API

    GGG provides an official Ladder API, however it has some serious drawbacks for real tools. The API is designed only to allow you to download the entire ladder (the top 15,000 players) and requires 75 GET requests to pull down the entire ladder.


    My Ladder API does all the work of pulling down the entire ladder from GGG every ten minutes, then makes the data available in a more reasonable way. This includes:

    • Request only specific character or account information
    • Show information calculated from deltas between updates (such as XP Gain, Online Time, etc.)
    • Search across multiple ladders for character or account information, including archived ladders
    • Request multiple accounts or characters in one request (for Guild tools or similar)
    • Show general league status, such as players on-line, levels, etc.
    • Request dumps of the entire ladder in modified JSON format (avoid doing this too often, it's heavy)

    Some tools that use my Ladder API include:

    • Ladder Track: A simple AHK GUI overlay showing your current rank status.
    • POE Simple Guild: A guild tool that uses my API to show last online status of guild members.
    • Guild Status: Another great guild tool that uses my API to show last online status of members.
    • Lutcikaur's Robust Macro and Overlay: Pulls JSON data to create a text overlay similar to Ladder Track, but with different data and performance.


    The Ladder API does not currently require or use API Keys, however I would appreciate it if you let me know that you're using it so I can add it to the list of tools!
